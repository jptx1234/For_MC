Spigot/Bukkit  Minecraft 服务端插件，实现for循环命令。


目前版本：for命令使用格式与大部分编程语言的for基本一致，被循环的命令支持如a*2、a++、sin(a)之类简单的表达式。用法：/for <变量名=初值>;[满足条件];[每次循环后执行的语句] <被循环的命令>


示例：/for i=1;i<=3;i++ setblock 0 4 i*2 stone


效果：在[0,4,2][0,4,4][0,4,6]处放置石头方块


目前暂只支持一行命令的循环，不能在循环体内运行多行指令。


目前不支持在服务器控制台和命令方块中运行for命令（考虑到不久会支持，现在懒得加判断限制条件，仅仅抛异常）
